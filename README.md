APC (Arbitrary Precision Calculator)
Overview

APC is a Data Structures and Algorithms project implemented in C that performs arithmetic operations on very large numbers beyond the limits of standard data types. The project uses linked lists to represent numbers digit by digit, enabling calculations on numbers of arbitrary length.

Features
Addition of large numbers
Subtraction of large numbers
Multiplication of large numbers
Division of large numbers
Dynamic memory management using linked lists
Supports numbers larger than built-in C data types
Technologies Used
C Programming
Linked Lists
Dynamic Memory Allocation
Learning Outcomes
Understanding linked list operations
Handling large numerical computations
Implementing arithmetic algorithms using data structures

Inverted Search
Overview

Inverted Search is a Data Structures and Algorithms project implemented in C that creates an inverted index for multiple text files. It allows users to search for words efficiently and identifies the files in which those words appear.

Features
Create an inverted index from multiple text files
Fast word searching using hashing
Display file information for searched words
Save and update indexed data
Efficient storage and retrieval of words
Technologies Used
C Programming
Hash Tables
Linked Lists
File Handling
Learning Outcomes
Understanding hashing techniques
Implementing efficient search algorithms
Working with file processing and indexing
